Регистр «НДС записи книги продаж»

Предназначен для хранения информации о записях книги продаж. По данным этого регистра строится отчет «Книга продаж» без всякой дополнительной обработки данных.

Информация хранится в следующем разрезе:

  • Покупатель;
  • Счет-фактура – первичный документ, которым был отражен начисленный НДС;
  • Вид ценности – косвенная информация о том, по какой причине был начислен НДС;
  • Ставка НДС;
  • Дата оплаты;
  • Документ оплаты;
  • Событие – одно из событий, связанных с начислением НДС;
  • Дата события – дата включения записи в книгу продаж;
  • Сумма без НДС;
